Consent

In this task, you will be presented with computer-generated images of groups of trees. For my study, I am interested in seeing how you perceive these images. The task will take 10-20 minutes.


On the next page you will see detailed instructions for the experiment.


By clicking "Continue" below, you agree to take part in this study. Data from this study will be kept confidential and only shown in aggregate form. This study is conducted in accordance with the ethical guidelines of KTH Royal Institute of Technology.


Instructions

In each trial of this experiment, you will be shown three images: one reference image, and two images for you to choose between. There will be 98 trials in total.


Your task is to choose which one of the two images you think contains the same ratio of trees as the reference image. On the next page, you will be shown an example.


Time limit: There is no time limit for the experiment, but for each trial, the images will be shuffled/updated if you haven't made a choice within 20 seconds.


If possible, please use a large monitor to do the experiment. If the page contents look like they don't fit the browser window at any point, please zoom the page out. Do this by pressing CTRL and - together. (If you are using a Mac, press Command instead of CTRL.)


Example

There are two kinds of trees used in each image.
Choose the image that you think contains the same ratio of different trees as the reference image.

Important: all three images will always contain the same total number of trees.
